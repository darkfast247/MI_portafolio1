Sobre Mí

¡Hola! Soy Santiago Loaiza

¡¡Bienvenido a mi portafolio! Soy un desarrollador de software apasionado por crear experiencias digitales increíbles. Mi enfoque principal se centra en el desarrollo backend; donde me destaco en el uso de tecnologías como .NET en C#, Kotlin y JAVA. en la gestión de bases de datos utilizando bases de datos SQL como MySQL y un poco de bases de datos NoSQL como MongoDB. Por la parte de frontend manejo de Angular como framework, Javascript como lenguaje, además de HTML y CSS. Estoy constantemente explorando nuevas formas de optimizar el rendimiento y la eficiencia en mis proyectos, buscando siempre ofrecer soluciones robustas y escalables para mis clientes. Mi objetivo es crear aplicaciones web funcionales y asegurarme de que ofrezcan una experiencia excepcional para los usuarios finales.
